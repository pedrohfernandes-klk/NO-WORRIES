# NO WORRIES

**A one-minute, self-contained audiovisual installation that runs entirely in a browser.**

No Worries is a shapeshifting HTML poem — a sixty-second piece about forced calm,
polite collapse, and the phrases we use to survive pressure. Text, sound, a countdown,
light, figures, search fragments, voices, and distortions unfold together in real time,
then reset. Press and hold **GO** and let it run.

## Run it

1. Open `index.html` in any modern browser (Chrome recommended), or host it anywhere static.
2. Press and **hold** GO to begin. The piece lasts ~60 seconds and ends on its own.
3. Press and hold **GO AGAIN** to replay.
4. The small **ⓘ** on the opening and final screens reveals the exhibition note.

No installation, no server, no build step.

## What's inside

- A **single, self-contained `index.html`** — no external CSS, JS, fonts, images, or audio files.
- **All audio is embedded** as base64; all artwork and code live inside the one file.
- **No frameworks, no dependencies.** It will keep working offline, from a USB stick, or projected on a wall.

## Format

HTML / sound / text / moving image — a live, browser-based installation built for gallery and projection contexts.

## Making-of

Built through successive human–machine iterations: writing, code, visual rhythm, sound design,
research, glitches, search fragments, and original voice recordings. The technical layers were
*vibe coded* — a newfound, instinct-led way of working. Made with human direction and computer minds
(ChatGPT and Claude, with NotebookLM for research and some support from Gemini). AI is treated as a
tool, not a cheat code.
